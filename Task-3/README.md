# Task 3: Customer Support Chatbot

## About the Task
In this task, a simple customer support chatbot was created to automatically respond to user queries such as contact details, pricing, and services.

## What Was Done
- Created a rule-based chatbot using Python
- Designed predefined questions and responses
- Built an interactive user interface using Streamlit
- Tested the chatbot in Google Colab environment

## Features
- Accepts user input through a text box
- Responds based on keywords like "contact", "price", and "service"
- Displays appropriate responses instantly

## Tools Used
- Python
- Streamlit
- Pandas

## Note
The Streamlit application runs successfully in Colab.  
Due to Colab environment limitations, the live web interface cannot be opened directly in a browser.
